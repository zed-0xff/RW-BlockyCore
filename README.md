# Blocky Core
[![Blocky Core](About/Preview.png)](https://steamcommunity.com/sharedfiles/filedetails/?id=2983955822)

This mod provides shared textures and functions for other Blocky mods.
Also adds 100+ props from the Minecraft.
"Blocky Props" architect menu is visible only in dev.mode by default, configurable in mod's settings.
